# Aruco Marker Placer

Приложение для размещения ArUco-маркеров на чертеже/изображении и экспорта проекта.

## Что умеет
- Загрузка фонового изображения.
- Загрузка DXF для автоматической калибровки масштаба.
- Выбор словаря ArUco, `id`, размера на фоне (`px`), реального размера (`mm`) и поворота (`yaw`).
- Авторасчет размера маркера в пикселях (`size_px`) из `size_mm` и `pixelCountPerMeter`.
- Размещение маркеров кликом по картинке или по ручному вводу `x/y` (в метрах).
- Импорт схемы из `json` формата как в `scheme(5).json`.
- Сохранение проекта в папку:
  - `scheme.json`
  - `layout.preview.png`
  - отдельные изображения маркеров: `id<id>_<size_mm>mm.png` и `id<id>_<size_mm>mm.svg`

## DXF + фон для автокалибровки
1. Загрузите фоновое изображение.
2. Загрузите DXF.
3. Нажмите `Калибровать ppm по DXF`.

Что происходит:
- Из DXF берется габарит modelspace (`X/Y`) с учетом `INSUNITS`.
- По габариту DXF и размеру фонового изображения вычисляется `pixelCountPerMeter`.
- Далее `size_px` автоматически пересчитывается из `size_mm`.

## Координаты
- В `json` координаты маркера (`x`, `y`) сохраняются в метрах.
- Нулевая точка: центр фонового изображения.
- `x`: вправо положительное.
- `y`: вверх положительное.

## Размер общей схемы
- При включенном `Авторасчет sizeX/sizeY` размер вычисляется автоматически.
- Используется максимум из:
  - метрического размера фонового изображения,
  - охвата всех размещенных маркеров с учетом их размера.
- Округление: вверх до `0.01 м`.

## Установка
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
```bash
python3 app.py
```

## Редактирование маркеров в списке
- Выберите строку в таблице и нажмите `Загрузить выбранный в форму` (или двойной клик по строке).
- Измените параметры (`id`, словарь, `x/y`, `size_mm`, `yaw`, при ручном режиме и `size_px`).
- Нажмите `Обновить выбранный маркер`.
