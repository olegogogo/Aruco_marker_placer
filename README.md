# Aruco Marker Placer

Приложение для ручного размещения ArUco-маркеров на чертеже/картинке с экспортом раскладки в YAML.

## Возможности
- Загрузка фонового изображения (чертеж с размеченными зонами).
- Выбор словаря ArUco (`DICT_6X6_250` и др.).
- Выбор `id`, размера маркера в пикселях на изображении и физического размера в мм.
- Расстановка маркеров кликом по изображению.
- Экспорт:
  - `layout.yaml` (координаты/размеры/словарь/ID).
  - `layout.preview.png` (визуализация с наложенными метками).
- Импорт ранее сохраненного `yaml`.

## Установка
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
```bash
python3 app.py
```

## Формат YAML
```yaml
version: 1
background_image: "/abs/path/to/background.png"
image_size_px:
  width: 1168
  height: 850
markers:
  -
    marker_id: 0
    dictionary: "DICT_6X6_250"
    cx_px: 350.0
    cy_px: 240.0
    size_px: 120
    size_mm: 130.0
    rotation_deg: 0.0
```

## Примечания
- Координаты `cx_px`, `cy_px` — центр маркера в пикселях исходного изображения.
- `size_px` — размер стороны маркера в пикселях в итоговой раскладке.
- `size_mm` — физический размер стороны маркера (для дальнейшей калибровки/производства).
